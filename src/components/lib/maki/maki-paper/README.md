### Paper

#### Props

| Name        | Type     | Default | Description          |
| ----------- | -------- | ------- | -------------------- |
| `elevation` | `Number` | `5`     | Shadow depth         |
| `rounding`  | `Number` | `3`     | The corner radius    |
| `stroke`    | `Number` | `0`     | The border thickness |

#### Slots

| Name      | Description                  |
| --------- | ---------------------------- |
| _default_ | The content of the component |
